# Informacje o projekcie 

## Podsumowanie
Projekt dotyczy strony www, która będzie hostowana na github i będzie stroną portfolio moich fotografii.

## Założenia

- Strona to statyczny html bez żadnych automatyzacji po stronie serwera.
- Strona ma mieć ładny przejżysty design, minimalistyczny - patrz przykłady niżej
- Strona ma mieć skrypt python służący do aktualizacji statcznych html 'build.py'
- Skrypt ma analizować podkatalog photos i na jego podsawie tworzyć galerie albo je aktualizować
- strona ma mieć jasny design i białe tło 
- w każdym podkatalogu w photos, będzie plik info.md który będzie tekstem dla galeriew jeżlei go nie ma, wstawmy tekst tymczasowy
- zdjęcia można kliknać i zostają pokazane nsamotnie na białym tle, i można przeskakiac do poprzeniego i następnego itd.
